\begin{abstract}
We study how autonomous AI agents can detect reliability degradation before
visible task failure, attribute root causes, estimate continuation risk, and
select recovery strategies. This draft accompanies the open-source
\texttt{reliable-agent-systems} repository.
\end{abstract}

\section{Introduction}
Capable agents must recognize when assumptions fail and choose when to retry,
adapt, ask for help, or stop. See the project roadmap for the full research plan.
